## What to Include

To help resolve your issue faster, please include:

- your device model
- iOS version
- PayGuard app version
- a short description of the issue
- steps to reproduce it, if possible

## Common Topics

### Restore Purchases
If you already bought Lifetime Pro and it is not unlocked, open PayGuard and use:

Settings > Restore Purchases

### Widgets
If widgets do not appear or update correctly:

- make sure you are using a supported iOS version
- try reopening the app once
- remove and add the widget again
- confirm that Lifetime Pro is unlocked if the widget is part of premium functionality

### Notifications
If reminders are not arriving:

- check notification permission in iPhone Settings
- make sure notifications are enabled for PayGuard
- confirm the record has reminders enabled inside the app

### Attachments and Imports
If a receipt, PDF, photo, or scan does not import correctly:

- try a clearer image or scan
- make sure the file is readable and not corrupted
- verify photo and camera permissions if needed
